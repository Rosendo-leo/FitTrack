FitTrack Android

App de treino, progresso físico e controle de peso — com sync, widgets e auto-update via GitHub.

Kotlin Jetpack Compose Room DB WorkManager Google Drive API Glance (Widgets) Retrofit GitHub Releases
Justificativa da stack
⚡
Kotlin + Jetpack Compose
UI moderna, declarativa, suporte oficial do Google. Melhor para Material 3 (dark theme, dynamic colors). Você já tem experiência, curva menor.
UI modernatipado
🗄️
Room + DataStore
Room para dados estruturados (treinos, séries, medidas). DataStore para preferências (tema, notificações). Tudo local, zero perda em updates.
localpersistente
☁️
Google Drive API
Backup automático em JSON/ZIP criptografado na pasta do app no Drive do usuário. Restore em um clique, portável entre dispositivos.
syncOAuth2
🔧
WorkManager
Agendamento confiável de notificações de treino e sync periódico com Drive. Sobrevive a reboots e battery optimization do Android.
backgroundrobusto
📦
GitHub Releases
Ao abrir o app, checa a GitHub API por nova versão. Se houver, exibe dialog de permissão antes de baixar e instalar o APK via PackageInstaller.
UPDATEpermission-first
🧩
Glance (Widgets)
Framework moderno para widgets Android usando Compose. Permite widgets interativos (registrar peso direto pelo widget) com UI consistente.
widgetinterativo

Fluxo de atualização via GitHub Releases
🚀
App abre
Dispara chamada à GitHub API: GET /repos/{user}/fittrack/releases/latest
🔍
Compara versões
Compara tag_name do release com BuildConfig.VERSION_NAME local usando semver. Se igual, silencioso.
💬
Dialog de permissão
Mostra dialog com: versão nova, release notes do GitHub, tamanho do APK, e botões "Atualizar agora" / "Depois". Nunca instala sem confirmação.
📥
Download + verificação
Baixa o APK com DownloadManager. Verifica SHA256 do asset antes de instalar (hash publicado no release). Progress bar no dialog.
✓
Instalação via PackageInstaller
Solicita permissão REQUEST_INSTALL_PACKAGES se necessário. Room DB e DataStore não são apagados — o APK só substitui o código.

Notificações
🏋️
Lembrete de treino
Horário configurável por dia da semana. "Hora do Treino A — Peito e Tríceps". Ação direta: abrir sessão.
⚖️
Registrar peso
Diária, horário fixo (ex: 7h30 ao acordar). Ação: abre dialog de registro diretamente sem abrir o app.
⏱️
Timer de descanso
Notificação persistente durante sessão ativa mostrando countdown. Vibra ao fim. Configurável por exercício ou global.
🔔
Nova atualização
Se o app estava em background quando encontrou update, notifica o usuário. Ao tocar, abre dialog de confirmação.
☁️
Backup realizado
Confirmação silenciosa após sync automático com Drive. Mostra data/hora e tamanho do backup. Configurável (pode desligar).
🏆
Novo PR!
Detectado automaticamente durante sessão. "Novo recorde: Supino 85 kg × 5". Celebração com vibração e som opcional.
Estratégia de backup e migração segura
🔒
Zero perda em updates
Room usa migrations versionadas. Ao atualizar schema, uma Migration class garante que os dados existentes são preservados e migrados automaticamente.
Room Migration
📤
Export manual
Exporta tudo em JSON comprimido (ZIP) para qualquer destino: armazenamento local, e-mail, Drive, etc. Inclui versão do schema para compatibilidade futura.
JSON + ZIP
☁️
Sync Google Drive
Automático em segundo plano via WorkManager (frequência configurável). Usa pasta oculta do app no Drive do usuário — não aparece no "Meu Drive".
auto-syncDrive API
📥
Restore inteligente
Ao restaurar, verifica conflitos entre dados locais e backup. Mostra dialog: substituir tudo / mesclar por data mais recente / cancelar.
mergesem perda
